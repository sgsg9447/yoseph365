| 항목 | 내용 |
| --- | --- |
| 문서 | 화면정의서 (Screen Definition) |
| 기준 | 확정 와이어프레임 + 데이터정의서 v2 + 정책 결정 |
| 표기 | 각 영역은 와이어프레임 번호(①②③…)와 1:1 매핑 |

## 0. 공통 정의 (All Pages)

| 영역 | 정의 |
| --- | --- |
| GNB | 로고 + 학원소개·과정 안내·국비지원·훈련 사진·상담문의 + [전화상담] 상주 |
| 전화상담 | 헤더 상주. 클릭 시 `tel:` 발신(모바일), PC는 번호·운영시간 노출 |

| 참조 | 데이터: 데이터정의서 v2 · 규칙: 정책정의서 |

## 정책·데이터 연계 체크

- 모집상태(모집중/마감/대기신청) → 정책정의서(수동 판정)
- 신청 유형 분기(A·B·C) → 데이터정의서 Course.fundingType
- 개인정보(이름·연락처) 수집 → 정책정의서 §8(동의·보유기간·접근권한)
- 취업률·개강일·잔여석 → 화면 미표시(결정 반영)
